This is a small ML project that uses scikit-learn Decision Tree Classifier Algo to tune the model.
https://scikit-learn.org/stable/modules/generated/sklearn.tree.DecisionTreeClassifier.html

Steps to follow:
I. Create a model ---- run model_generator.py:
    1. Import Data
    2. Clean the Data
    3. Split the data into training and test. (80:20)
    4. Create a Model using algos
    5. Train the model
    6. Make Predictions
    7. Evaluate and Improve

II. Load the model and test ---- run main.py
